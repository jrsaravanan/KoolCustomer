# KoolCustomer

A Microservice PoC. 

Build Using
* JDK 8
* Spring Boot
* Spring Admin
* Eureka
* Admin UI
* Cucummber JVM
* Rest Assured
* REST Doc

## Build 
```
$ .\mvn.install.sh
$ docker-compose up
```
